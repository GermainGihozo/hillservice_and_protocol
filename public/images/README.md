## 📏 Photo Specifications:

- **Format**: JPG or PNG
- **Size**: 800x600 pixels (recommended)
- **Ratio**: 4:3 (landscape orientation)
- **File Size**: Under 500KB (optimized for web)
- **Quality**: High resolution, clear and professional

---

## ✅ After Adding Photos:

1. Edit this file: `src/components/Gallery.jsx`
2. Replace image URLs with: `/images/your-photo-name.jpg`
3. Update titles and descriptions
4. Save and refresh your browser!

---

## 🎨 Example:

If you add a file named `wedding-1.jpg` here, use it in code like this:

